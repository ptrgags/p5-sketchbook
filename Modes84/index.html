84 Musical Modes (2026-04)

This sketch is a demonstration for my other project, pattern-spotting, which is all about documenting patterns in music, art, and math. More specifically, the page on modes of the major scale describes this pattern.

In short, there are 7 modes of the major scale in music. Each mode can start on any of the 12 pitches on the piano keyboard. That means there are 7 * 12 = 84 unique modes. With a careful ordering (see the link above), you can cycle through all 84 modes, adding a single flat to the scale each time.

Some observations: You start at C Lydian which is bright, and it slowly sounds darker as you add flats. However, after C Locrian, the next flat lowers the C to a B, and the scale happens to be B Lydian. This has an interesting sound, as we're both lowering the root by a half step, but also jumping from the darkest mode (Locrian) to the brightest one (Lydian).
