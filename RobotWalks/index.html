Robot Walks (2025)


Use left/right arrows (or touch the left/right sides of the canvas) to move the robot. Use the buttons below the canvas to switch between robot turn values.

This sketch is an interactive version of the printable artworks I made in my paper-toaster project. The Paper Toaster sketch is in turn inspired by Project Euler #208: Robot Walks. The goal is not to solve the original math puzzle, but rather to visualize the patterns.

In the original problem, the robot moves in arcs that are 1/5 of a circle. This sketch allows you to select between a few different turn angles. For example, a 4-robot makes 1/4 turns. This addition was inspired by this article by Peter Kagey.

Depending on the robot's path, sometimes you get satisfying squiggly patterns, other times you get a chaotic pile of overlapping arcs. If you're careful, you can get the robot to make an intricate loop, or patterns with rotation symmetry like a flower.
