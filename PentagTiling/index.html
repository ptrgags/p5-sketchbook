Pentag Tiling (2024)

Click any tile with a circle to add arcs to the tiling.

It is possible to get stuck (i.e. have a tile with no valid options). One strategy to prevent this is to always click through an entire column or row at a time, progressing across the grid in a single direction.

This is a tiling of tag-shaped pentagons (I call them "pentags" for lack of a better word) and arcs. It's based on some doodles I made on graph paper.

For more details, see the explainer.
